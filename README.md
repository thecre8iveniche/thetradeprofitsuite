# TradeProfit Suite

**All-in-one estimating, job costing, and invoicing tool for contractors and tradespeople.**

Stop losing money on underbid jobs, forgotten change orders, and invoices that don't match what a job actually cost. TradeProfit Suite connects your estimate, materials, actual costs, change orders, and invoice into one job record — so nothing falls through the cracks.

🔗 **[Try the live app](https://github.com/thecre8iveniche)** 

---

## Features

- **Estimate Calculator** — line items, labor, markup %, overhead, and tax, all rolled into one running total
- **Material Calculator** — figure quantities for area, linear, or count-based materials with waste % built in
- **Job Cost Tracking** — log actual materials, labor, and other costs against the estimate to catch overruns early
- **Change Order Log** — track scope changes with date, description, and amount; approved changes flow into the invoice automatically
- **Invoice Generator** — a clean, printable invoice built from the estimate and approved change orders
- **Profit Tracking Dashboard** — see what every job actually made, across your whole job list
- **Customer Records** — save contact info and job site details for every customer
- **Backup & Restore** — export all data to a file and restore it anytime, on any device

## How it works

TradeProfit Suite runs entirely in the browser — no server, no account, no subscription. Data is saved locally using the browser's storage, so it stays private on the device it's used on. The built-in Backup feature lets users export their data to a file and restore it elsewhere.

**Tech:** plain HTML, CSS, and JavaScript — no build step, no dependencies, no framework. This keeps it fast, portable, and easy to host anywhere that can serve a static file (GitHub Pages, Netlify, or a plain web server).

## Running it locally

No installation needed. Clone or download this repo and open `index.html` in any modern browser, or serve the folder with any static file server.

```
git clone <this-repo-url>
cd <repo-folder>
open index.html
```
